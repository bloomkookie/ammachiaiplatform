# 🌾 Krishi Sakhi - Digital Farming Companion

**Your Intelligent Agricultural Assistant for Kerala Farmers**

Krishi Sakhi is a comprehensive digital farming platform designed specifically for Kerala farmers, providing AI-powered agricultural guidance, real-time market data, and personalized farming advice in both English and Malayalam.

- **Malayalam Voice Support** - Text-to-speech and voice interaction
- **Contextual Recommendations** - Weather-based farming guidance
- **Smart Fallback System** - Works even when AI APIs are unavailable

- **Agricultural Officers** - Book consultations with experts
- **Professional Booking** - Detailed confirmation and scheduling
- **Direct Contact** - Phone and video consultation options

### 🎨 Modern User Experience
- **Beautiful UI** - Professional design with smooth animations
- **Responsive Design** - Works on all devices

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ and npm
- Python 3.8+ and pip
- Git

### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd ammachi_ai-main
```

2. **Setup Frontend**
```bash
cd frontend
npm install
npm run dev
```

3. **Setup Backend**
```bash
cd ../ammachi_django_backend
python -m venv .venv_django
source .venv_django/bin/activate  # On Windows: .venv_django\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver 8001
```

4. **Configure Environment Variables**
```bash
# In ammachi_django_backend/.env
GEMINI_API_KEY=your_gemini_api_key_here
PLANT_ID_KEY=your_plant_id_api_key_here
OPENWEATHER_API_KEY=your_weather_api_key_here
```

### 🌐 Access the Application
- **Frontend**: http://localhost:5174
- **Backend API**: http://localhost:8001
- **API Documentation**: http://localhost:8001/api/

## 🔧 Configuration

### API Keys Required
1. **Gemini API Key** - Get from Google AI Studio
2. **Plant.id API Key** - Get from Plant.id website
3. **OpenWeather API Key** - Get from OpenWeatherMap

### Environment Setup
Create `.env` file in `ammachi_django_backend/`:
```env
GEMINI_API_KEY=your_gemini_api_key
PLANT_ID_KEY=your_plant_id_key
OPENWEATHER_API_KEY=your_weather_key
DEBUG=True
SECRET_KEY=your_django_secret_key
```
